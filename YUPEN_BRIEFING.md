## 업데이트 방식
### 일별 동향 추가
- context.md 맨 끝에 cat >> 로 append
- 형식 예시:
  ### 5/8 (금) 부산 동향
  - 항목1
  - 항목2

### 북구갑 여론조사 추가
- 섹션 3 끝부분에 추가
- 형식: 기관명/날짜/n수/방법 + 정당지지도 + 다자대결 + 양자대결 + 핵심변화

### 부산시장 여론조사 추가
- 섹션 10 끝부분 또는 맨 끝에 추가
- 형식: 기관명/날짜/n수 + 후보 지지율 + 핵심 해석

## 모든 명령어는 클로드 코드 터미널에 입력
- 파일 확인: cat, sed, tail, grep
- 내용 추가: cat >> 파일경로 << 'EOF' ... EOF
- 줄 수정: sed -i 명령어 (Windows는 주의)
