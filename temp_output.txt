### 두 조사 종합 해석 및 북구갑 시사점

**부산시장 판세**
- 두 조사 모두 전재수 43~44% 대 박형준 39~44% 초박빙
- 바로미터: 전재수 +4.7%p 우세 / 제이투: 전재수 +0.2%p 사실상 동률
- 3권역(북구 포함)에서 전재수 우세 (45~50% vs 35~43%)

**북구갑 분리투표 시나리오**
- 시나리오 A (민주 쏠림): 부산시장 전재수 + 국회의원 하정우 → 러닝메이트 효과
- 시나리오 B (분리투표): 부산시장 전재수 + 국회의원 한동훈 → 중도층 분리투표
- 시나리오 C (보수 결집): 부산시장 박형준 + 국회의원 보수단일화후보 → 보수 완전결집

**연령대별 분리투표 가능성**
- 40~50대: 전재수 강세(52~56%) + 하정우 강세(36~46%) → 민주 동반 강세
- 60대: 박형준 우세(44~50%) + 하정우 40% → 분리투표 가능성 높은 세대
- 70대+: 박형준 압도(55~56%) + 한동훈 35% → 보수 결집 세대

**가덕도 이슈 (북구갑 공약 시뮬레이션 참고)**
- 민주당 지지자도 가덕도 완공보다 김해공항 확장 선호
- 공항 이슈는 진보·보수 구분 없이 현실론 우세
- 하정우가 가덕도 완공 주장 시 역풍 가능성 있음

### 여론조사꽃 부산시장 CATI (4/28~29, n=1,002, 응답률 13.5%) ? 3차 조사

**조사 설계**
- 조사방법: 무선가상번호 CATI 전화면접 (SKT/KT/LGU+ 전체 29,983개)

- 3권역(북구·사하구·강서구·사상구) ? 북구갑 분리투표 분석에 핵심

**정당지지도 (부산 전체, 재질문 포함)**

- 3권역(북구 포함): 민주 51.4% / 국힘 29.9% ? 북구 민주 우세 확인
- 40대: 민주 67.4% / 50대: 66.9% (민주 압도)
- 70대+: 국힘 54.6% / 민주 34.0% (국힘 우세)

**이재명 국정운영 평가 (부산 전체)**
- 잘함 71.8% / 못함 26.3% / 모름 2.0%
- 3권역(북구 포함): 잘함 75.6% ? 북구갑 페르소나 시뮬레이션 핵심 수치
- 40대: 잘함 88.1% / 50대: 84.4% / 60대: 61.1% / 70대+: 60.6%
- 보수층: 잘함 45.4% / 중도층: 77.8% / 진보층: 96.2%

**지방선거 인식**
- 여당(이재명 정부) 지지 56.1% / 야당 지지 35.4%
- 3권역: 여당 56.1% / 야당 35.1%
- 40대: 여당 72.9% / 50대: 71.7%
- 70대+: 야당 49.0% > 여당 39.1% (역전)

**부산시장 다자대결 ? 전재수 vs 박형준 vs 정이한**
- 전재수 49.8% / 박형준 28.5% / 정이한 1.8% / 없음+모름 19.9%
- 3권역(북구 포함): 전재수 51.0% / 박형준 25.1%
- 40대: 전재수 70.0% / 50대: 72.9% (압도)
- 60대: 전재수 43.9% / 박형준 38.1% (접전 ? 분리투표 가능 세대)
- 70대+: 박형준 50.3% > 전재수 32.1% (역전 ? 보수 결집)
- 이재명 잘함층: 전재수 68.6% / 이재명 못함층: 박형준 68.8%

**부산시장 양자대결 ? 전재수 vs 박형준**
- 전재수 52.8% / 박형준 30.5% / 없음+모름 16.5%
- 3권역: 전재수 55.4% / 박형준 28.3%
- 진보층: 전재수 86.2% / 중도층: 전재수 58.9% / 보수층: 박형준 63.6%

**북구갑 분리투표 시사점**
- 북구 포함 3권역에서 전재수 51~55% 우세 ? 하정우 지지와 연동 강함
- 이재명 잘함층의 전재수 지지 강력 → 하정우·전재수 동반지지 가능성
- 60대: 전재수·박형준 접전 → 국회의원 선택에서 분리투표 발생 가능
- 70대+: 박형준 우세 → 한동훈·박민식 지지와 연동 ? 보수 결집 세대
- 중도층이 전재수 58.9%로 쏠림 → 북구갑 하정우 중도층 확보 가능성
