## 5. 일별 부산 동향 (5/1~)

### 5/1 (목) 부산 동향
- 하정우, '손 털기' 논란 해명 ("손이 저렸던 것") — 韓·朴 측 '오만' 프레임으로 집중 공세
- 한동훈, 팬덤·전국 화제성 + 북구 주민 중심 선대위 '지역 밀착' 양면 전략 병행
- 이영풍 전 KBS 기자, 북갑 출마 선언 — 보수 후보 3파전 구도
- 보수 단일화 여부 최대 변수로 부각
- 부산시장: 민주 'HMM 부산 이전·전재수 성과' vs 국힘 '낙동강 전선 사수·보수 결집'
- 장동혁 지도부, 박형준 선대위 개소식 총출동

### 5/4 (월) 부산 동향
- 전국 여당 우세 흐름 속 PK는 보수 결집·접전 양상 — 북구갑이 최대 변수
- 하정우, 북구 전입신고 후 구포시장 방문 — '지역 밀착' 행보
- 한동훈, 예비후보 등록 앞두고 구포시장 민심 잡기
- 국힘, 박민식·이영풍 경선 후 단일화 예정 — 보수 단일화가 판세 핵심
- 전재수, HMM 부산 이전 성과·'집권여당 실행력' 강조
- 박형준, 장동혁 지도부 지원 속 글로벌허브도시특별법 책임론으로 맞대응

### 5/5 (화) 부산 동향
- 하정우, '손 털기' + 정청래 '오빠' 발언 논란 — 정치 초보 리스크 부각
- 한동훈, 예비후보 등록 완료 — "만덕·덕천·구포 과제를 부산 1순위로" 지역 밀착 메시지
- 북갑, 완전한 박빙 구도로 전환 확인
- 민주당, 부산 현장최고위 + PK 공천자대회 — 전재수·하정우 '원팀' 구도 공식화
- 전재수, 박형준 대형사업 재검토·'민생 100일 비상조치' 1호 공약 제시
- '공소 취소 특검법' 논란 — 야권 공동 대응, 민주 내부 처리 시점 갈등

### 5/6 (수) 부산 동향
- 박민식, 국민의힘 후보 확정 — "한동훈과 단일화 가능성 제로" 선 긋기
- 보수표 분열 현실화 — 하정우·한동훈·박민식 3자 대진표 확정
- 최근 여론조사 하정우·한동훈 오차범위 내 초박빙 — 압도적 우세 후보 없이 예측불허
- 부산시장: 전재수·박형준 격차 좁혀짐 — 청년·60대·무당층이 캐스팅보트
- 하정우, "현실정치 처음이라 소통 미숙했다 — 국회의원에게 필요한 건 이념이 아닌 능력" 해명

- 朴 "단일화는 정치공학" — 완주 의사 재확인, 보수 분열 고착화
- 與 전재수·하정우 'AI도시 부산' 공동 공약 — 북갑+부산시장 묶는 연계 전략
